# Python based Password generator

## Installation
### Requirements
```powershell
PS C:\Project> pip install -r requirements.txt
```
### Execute using python
```powershell
PS C:\Project> python main.py
```
### Export as an executable 
```powershell
PS C:\Project> pyinstaller --onefile --windowed -n passwordGen.exe main.py
```
Exported file can be found under dist/passwordGen.exe
